* Each job contains **steps**, which can:

### âœ… Basic Example:

```yaml
jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - name: Checkout code
        uses: actions/checkout@v3

      - name: Set up Node.js
        uses: actions/setup-node@v3
        with:
          node-version: 18

      - name: Install dependencies
        run: npm install

      - name: Run tests
        run: npm test
```

## ğŸ” Sequential Jobs with `needs`

You can run jobs in order using `needs:`:

```yaml
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - run: echo "Build step"

  test:
    needs: build
    runs-on: ubuntu-latest
    steps:
      - run: echo "Test after build"
```
