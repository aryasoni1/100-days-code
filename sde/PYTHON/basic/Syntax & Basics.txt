# ✅ 1. Python Fundamentals (All Roles)

> 🧠 **Master the building blocks of the Python language** to form a strong foundation — useful across data science, web dev, automation, and more.

---

## 🧱 Syntax & Basics

### 🔤 **Variables**

Used to store data in memory.

```python
name = "Alice"
age = 25
```

Python is **dynamically typed** — you don’t need to declare types explicitly.

---

### 🔢 **Data Types**

| Type    | Example         |
| ------- | --------------- |
| `int`   | `10`            |
| `float` | `3.14`          |
| `str`   | `"hello"`       |
| `bool`  | `True`, `False` |
| `list`  | `[1, 2, 3]`     |
| `dict`  | `{"a": 1}`      |
| `set`   | `{1, 2, 3}`     |
| `tuple` | `(1, 2)`        |

Use `type()` to check data type:

```python
print(type(age))  # <class 'int'>
```

---

### ➕ **Operators**

| Type       | Operators                           |
| ---------- | ----------------------------------- |
| Arithmetic | `+`, `-`, `*`, `/`, `%`, `//`, `**` |
| Comparison | `==`, `!=`, `>`, `<`, `>=`, `<=`    |
| Logical    | `and`, `or`, `not`                  |
| Assignment | `=`, `+=`, `-=`, etc.               |

Example:

```python
x = 5
y = 10
print(x > 3 and y < 20)  # True
```

---

### 🔁 **Conditional Statements**

Control the flow of logic.

```python
age = 18

if age >= 18:
    print("Adult")
elif age > 12:
    print("Teenager")
else:
    print("Child")
```

> 📝 Indentation is **mandatory** in Python. No `{}`!

---

## ✅ Summary

| Concept      | Why It Matters                                   |
| ------------ | ------------------------------------------------ |
| Variables    | Store and manipulate data                        |
| Data Types   | Understand what kind of data you’re working with |
| Operators    | Perform operations on values                     |
| Conditionals | Decision-making logic                            |
