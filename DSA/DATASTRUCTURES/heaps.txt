A **Heap** is a special tree-based data structure that satisfies the heap property:

* **Min-Heap**: Parent â‰¤ Children (smallest at the root)
* **Max-Heap**: Parent â‰¥ Children (largest at the root)

Heaps are typically implemented using arrays for space and time efficiency.

## âš™ï¸ Core Operations

| Operation | Description                | Time Complexity |
| --------- | -------------------------- | --------------- |
| `push`    | Add element to heap        | O(log n)        |
| `pop`     | Remove root element        | O(log n)        |
| `heapify` | Convert array to heap      | O(n)            |
| `peek`    | View root without removing | O(1)            |

### Heapify Example


## ğŸ¯ Applications

### ğŸ¥‡ Priority Queue

* Elements with higher priority are served first.
* Built-in `heapq` can be used for efficient priority queues.

### ğŸ“Š Heap Sort

```python
def heap_sort(nums):
    heapq.heapify(nums)
    return [heapq.heappop(nums) for _ in range(len(nums))]
```

* Time: O(n log n)
* Space: O(1) (in-place possible)

### ğŸ” Kth Smallest/Largest Element

```python
def kth_smallest(nums, k):
    heapq.heapify(nums)
    for _ in range(k - 1):
        heapq.heappop(nums)
    return heapq.heappop(nums)

def kth_largest(nums, k):
    max_heap = [-x for x in nums]
    heapq.heapify(max_heap)
    for _ in range(k - 1):
        heapq.heappop(max_heap)
    return -heapq.heappop(max_heap)
```
